# sec-mind
记录整理、实践的安全场景脑图。

方便为不同体系进行归纳，将全部脑图主要六个目录，分别为
- 漏洞挖掘&渗透测试`VulExploit&PT`
- Web安全`WebSecurity`
- 代码审计`CodeAudit`
- 安全运维`SecOps`
- 安全能力建设`SecCapacityBuilding`
- 相关工具`Tools`
